## 📖 About RefRed

**_RefRed_** is a **referrer-based redirection** service designed to streamline **relative linking** across the web, particularly in contexts where it isn't natively supported. It dynamically redirects users to paths resolved against the referrer's domain, so a single link can behave differently depending on where it was clicked from. A common example is _GitHub_ markdown — especially default community health files used across many repositories — where absolute links can't point to "the current repo" and relative ones don't travel consistently across forks and mirrors.
